# 开发规范
***

- 1、类注释  
  类、接口、枚举前必须要添加自动注释，参考如下
  ```java
  #if (${PACKAGE_NAME} && ${PACKAGE_NAME} != "")package ${PACKAGE_NAME};#end
  #parse("File Header.java")
  
  /**
  * ProjectName: ${PROJECT_NAME}
  * CreateTime: ${DATE} ${TIME}
  * Author: vincentEnxy
  * Version: 1.0
  * Description:
  */
  public class ${NAME} {
  }
  
  ```
